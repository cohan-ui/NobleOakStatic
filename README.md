# NobleOak Static — Life Cover Check-in screens

Frozen, script-free HTML snapshots of each prototype screen (414px, filled
representative state) for importing into Figma via html.to.design.

- `index.html` — directory of all screens
- `01-welcome.html` … `10-done.html` — the 10-screen journey
- `11-coach-sheet-open.html` — Life Cover Coach ask-mode state

Generated by `tools/export-figma-pages.js` in the main prototype repo.
Do not hand-edit; regenerate after visual changes.
